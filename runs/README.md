# Optimization Runs

This directory stores immutable optimization run records generated by `optimization_infra`.

Expected layout:

```text
runs/
  <run_id>/
    config_snapshot/
    EXECUTION_PLAN.md
    execution_plan.yaml
    observations.yaml
    SCAN_SUMMARY.md
    scan_summary.yaml
    DECISION_PACKET.md
    decision_packet.yaml
```

Scientific outputs may live under a run-specific `outputs/` directory when a candidate is actually executed. Infrastructure dry runs may contain only plans, registry entries, and summaries.

Do not treat a dry-run plan as a completed scientific run.
